# <h1>URL Formatter</h1>

URL Formatter build with TypeScript, React and MongoDB

<p>Server repository <a href="https://github.com/dauphaihau/url-formatter-server">here</a>. </p>

## Install Client (React Files)

Install dependencies via npm or yarn
```shell
yarn
```

Start client
```shell
yarn start
```
